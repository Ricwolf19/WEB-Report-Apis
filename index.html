Principales Conceptos

Servicio Web

Un servicio web es una unidad lógica de funcionalidad que se expone a través de la web y se puede invocar mediante protocolos estándar como HTTP. Los servicios web suelen utilizar descripciones de servicios, como WSDL, para definir su interfaz.

SOAP (Simple Object Access Protocol)

SOAP es un protocolo de comunicación utilizado en servicios web para intercambiar mensajes estructurados en formato XML.

REST (Representational State Transfer)

REST es un estilo arquitectónico que utiliza HTTP y se basa en la representación de recursos. Los servicios RESTful son una alternativa a SOAP y utilizan solicitudes HTTP como GET, POST, PUT y DELETE.
